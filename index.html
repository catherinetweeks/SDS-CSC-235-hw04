Official and Unofficial Native Languages Spoken in Each Country Around the World

This tool visualizes official and unofficial native languages spoken in each country around the world from a dataset of global language information from Kaggle. Each point represents a language, its size the percent of native speakers in its corresponding country, and its color its official status in said country. The map can be zoomed in either by using mousepad gestrues or by using the zoom buttons in the bottom right corner. The position can be altered by clicking and dragging. Language names can be shown when hovered on, and can also be clicked on to display a bar graph of native speaker percentages.
